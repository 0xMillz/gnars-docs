## If you wanna run localy like a pro: 

Fork it, or not

```
git clone https://github.com/sktbrd/gnars-docs.git
cd gnars-docs
npm run start
```

You gotta have [Node.Js](https://nodejs.org/en/) and know some shit, or you can just be gnarly about it
